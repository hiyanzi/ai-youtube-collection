AI YouTube 影音知識庫

獨立全端開發者專案 | AI-Powered YouTube Knowledge Base

Python React Tailwind CSS Gemini 2.5 Flash API YouTube Data API Vercel
背景與挑戰 (Situation & Task)

為了解決 YouTube 播放清單中收藏了上百支 AI 教學影片卻難以分類、過濾與消化的高學習門檻，決定打造一套全自動化的知識管理工具。核心挑戰在於如何將大量且非結構化的影片內容，轉化為客觀可搜尋的量化指標，藉此快速判斷每支影片的真實學習價值。

解決方案與行動 (Action)
  • AI 自動化解析管線： 開發 Python 腳本串接 YouTube API 獲取影片元數據，並創新地利用 Gemini 2.5 Flash 直接讀取影片 URL 進行多模態深度分析，成功省去建立複雜字幕爬蟲工程的開發成本。
  • Prompt 系統工程優化： 針對 LLM 容易產生幻覺 (Hallucination) 與過度解讀的現象，設計了嚴格的 Prompt 條件。例如：規定影片需專門討論特定技術 30 秒以上才可計分，成功將非結構化的內容轉化為客觀的「落地度評分 (0-10)」與「技術標籤」。
  • 系統穩定與容錯設計： 針對長時間執行的批次 API 請求，自主實作了斷點續傳機制，有效避免因網路中斷或免費 Quota 用盡而流失資料，確保資料處理任務具備 Production 等級的穩定性。
  • 輕量化無後端前端架構： 採用 Serverless 的設計理念，前端以單一 React 檔案結合 CDN 與 localStorage 實現使用者狀態持久化，不僅達成「極致輕量」，更確保了系統長期零維護成本與快速的讀取效能。
專案成果 (Result)

成功將 13 個零散、無組織的播放清單整合為一個具備「跨欄位關鍵字搜尋」與「三層維度篩選」的高效知識庫網站。透過流程與模型調優，將單支影片的 AI 處理成本極致壓縮至不到 $0.03 美元，大幅提升了個人對新技術的學習與檢索效率，並成功透過 Vercel 實現自動化 CI/CD 部署上線。
